Age and Gender Recognition Using CNN

HMC CS152 Final Project by Olivia Watkins and Jasmine Zhu

Introduction

The problem is recognizing human age using deep learning approch consists of image classification and softmax expectation rather than classic regression approach.

Dataset and Setup

We use the IMDB-WIKI data set, which contains 500k+ faces images with age and gender labels scraped from IMDB and Wikipedia. It is the largest labeled age dataset available online. It is unblanced with a concentratd distribution of age between 20 and 40. All pictures in this dataset are raw pictures scraped from websites and need pre-processing. To pre-preocess the images, we use dlib to align the faces. This library will return four coordinates of cropped face box. We choose to keep 40% margin of cropped faces because this attempt preserves more context information which improve their results accroding to the experiment in IMDB-WIKI paper. After that, we resize the faces to 224x224 to fit the size of our convolutional neural network. Due to the limitation of hardware, we only use the WIKI datasets which contains 62,359 pictures.

We implement our model using Keras and Tensorflow library. We've run on two GPUs. One is Bridges supercomputer at the Pittsburgh Supercomputing Center (PSC). The other one an Amazon EC2 g2.2x large GPU using community image vict0sch, with 10GB memory.
